### 5.1节：新增登陆页面路由

在系列3章节中，我们引入了路由对象，定义了路由跳转，那么这次，我们对之前的代码稍作调整，将登陆路由跟首页路由放在相同的层级上，调整后的代码如下

```javascript
// 定义路由跳转
const routes = [
    {
        name:'home',
        path:'/',
        meta:{
            title:'首页'
        },
        component:Home,
        // 重定向
        redirect:'/welcome',
        children:[
            {
                name:'welcome',
                path:'/welcome',
                meta:{
                    title:'欢迎'
                },
                component:Welcome
            }
        ]
    },
    {
        name:'login',
        path:'/login',
        meta:{
            title:'登陆'
        },
        component:Login
    }
]
```
